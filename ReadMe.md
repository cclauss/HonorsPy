## About The Project
It's a search engine that gives you control for your privacy, no ads, no bias results. Just results.


### Built With
- [ ] Python
- [ ] Django
- [ ] HTML
- [ ] CSS
- [ ] Bootstrap
- [ ] JS

### Prerequisites

* Python
* Django
* Docker
* Docker Compose

### Installation
1. docker-compose build
2. docker-compose up
3. Go to `http://0.0.0.0:8000/`, that's where search engine will be.
